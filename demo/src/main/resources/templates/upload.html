Загрузка файлов с возможностью поиска строки внутри загруженных файлов

Описание работы:

Файлы сохраняются в System.getProperty("java.io.tmpdir").
Файл разбивается на строки, строки разбиваются на слова (split ' '), слова являются ключами в Map, значением является список координат слова - (файл, строка, позиция в строке)

API:

/search?string=search_str

Описание работы: search_str разбивается на слова (split ' ') происходит поиск координат каждого слова с точностью до строки, определяются те строки которые были найдены количество раз = количеству слов в search_str внутри этих строк делается поиск через String.contains(search_str) если подcтрока найдена - она возвращается в качестве результата
